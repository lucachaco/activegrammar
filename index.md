| 2. Person     | du                    | ihr   |
|               | Sie(Höflichkeitsform) | Sie   |

## Konjugation Präsens I

|               | Singular       | Plural             |
| ------------- |:---------------| :------------------|
| 1. Person     | ich geh-**e**  | wir geh-**en**     |
| 2. Person     | du geh-**st**  | ihr geh-**t**      |
|               | Sie geh-**en** | Sie geh-**en**     |
| 3. Person     | er geh-**t**   | sie geh-**en**     |
|               | sie geh-**t**  |                    |
|               | es geh-**t**   |                    |
